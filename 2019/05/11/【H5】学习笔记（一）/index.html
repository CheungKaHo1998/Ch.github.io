【H5】学习笔记（一）

什么是HTML5

他是html下一代标准，还在完善之中，大部分浏览器都支持（ Internet Explorer 8 及更早 IE 版本的浏览器不支持）

自定义元素

Internet Explorer 8 及更早 IE 版本的浏览器不支持的问题

新多媒体元素

audio 定义音频内容
video 定义视屏
embed 定义嵌入内容，比如插件
track 添加文本轨道

常用新元素

progress 定义任何类型的任务的进度
nav 定义链接
mark 定义带有记号的文本
header 定义了文档的头部区域
footer 定义页脚
canvas 绘制图形，通过js来完成
drag&drop 拖放（案列w3c）
geolocation 地理定位

新的Input类型

color 从拾色器中选颜色
datatime 日期
email 邮箱
number 数字
search 搜索域
tel 号码
url 地址
