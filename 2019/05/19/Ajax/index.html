Ajax

Ajax

Ajax即“Asynchronous Javascript And XML”（异步JavaScript 和XML）
Ajax的主要优势就是对页面的请求以异步方式发送服务器
而服务器不会用整个页面来响应请求，它会在后台处理请求
与此同时还能继续浏览页面并与页面交互。

XMLHttpRequest对象

Ajax技术核心就是XMLHttpRequest对象，这个对象充当着浏览器中的脚本（客户端）与服务器之间的中间人的角色。以往的请求都有浏览器发出，而JavaScript通过这个对象可以自己发送请求，同时自己处理响应。

getHTTPObject

通过对象检测技术检测XMLHttpRequest，如果失败则检测其他方法，最终返回false或一个行的XMLHttpRequest（或XMLHTTP）对象。
在脚本中要使用XMLHttpReques对象时，将这个新对象直接赋值给一个变量：

var request=getHTTPObject();

open方法

它用来指定服务器上将要访问的文件，指定请求类型：GET、POST和SEND。着三个方法的第三个参数用来指定请求是否以异步方式发送和处理。
